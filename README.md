# Lesson 1.4 – Statisk meny

Detta steg introducerar en enkel **statisk meny** i HTML.

Menyn är inlagd högst upp på varje sida och består av:

- en `<nav>`-tagg
- en lista (`<ul>`)
- länkar (`<a>`)

Syftet är att:
- förstå hur en meny byggs med ren HTML
- se att samma kod behöver upprepas på varje sida
- förbereda nästa steg där menyn byggs automatiskt med JavaScript
